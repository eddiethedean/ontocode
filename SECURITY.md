# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.21.x  | Yes — latest tagged release |
| 0.20.x  | Yes |
| 0.19.x  | Yes |
| 0.18.x  | Yes       |
| 0.17.x  | Best effort |
| 0.16.x  | Yes       |
| 0.15.x  | Yes       |
| 0.14.x  | Best effort |
| 0.13.x  | Best effort |
| 0.12.x  | Best effort |
| 0.11.x  | No        |
| 0.10.x  | Best effort |
| ≤ 0.9.x | No        |

Pin production and CI to the latest **tagged** release (`docs/TAGGED_RELEASE` in the repository). Unreleased minors on `main` are not supported for adoption until tagged.
