# Support

Community support for OntoCode / OntoCore lives in the documentation and GitHub Discussions.

**Canonical page:** [docs/support.md](docs/support.md) · [Read the Docs](https://ontocode-vs.readthedocs.io/en/latest/support/)

| Channel | Use for |
|---------|---------|
| [GitHub Discussions](https://github.com/eddiethedean/ontocode/discussions) | Questions, how-to, evaluation |
| [GitHub Issues](https://github.com/eddiethedean/ontocode/issues) | Bugs and actionable feature requests |
| [Security policy](SECURITY.md) | Vulnerability reports |

Before opening an issue, check [Troubleshooting](https://ontocode-vs.readthedocs.io/en/latest/troubleshooting/) and [FAQ](https://ontocode-vs.readthedocs.io/en/latest/faq/).
