# ontocore-diagnostics

> Part of **OntoCore** (semantic workspace engine).

Ontology lint rules and diagnostic collection for [OntoCore](https://github.com/eddiethedean/ontocode).

## Install

```toml
ontocore-diagnostics = "0.13"
```
