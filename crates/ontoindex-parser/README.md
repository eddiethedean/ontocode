# ontoindex-parser

RDF parsing and entity extraction for [OntoIndex](https://github.com/eddiethedean/ontocode), backed by [Oxigraph](https://crates.io/crates/oxigraph).

## Install

```toml
ontoindex-parser = "0.6"
```
