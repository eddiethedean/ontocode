# WebProtégé Features

## Purpose
This document catalogs the major capabilities of WebProtégé and identifies modernization opportunities for OntoCode.

## Core Features
- Browser-based ontology editing
- Shared projects
- Entity editing
- Class, property, and individual management
- Comments and discussions
- Change history
- Search
- Reasoner integration
- Project sharing
- Tagged releases

## Modernization
- Offline-first support
- AI-assisted modeling
- Git-native workflows
- Real-time graph editing
- Rich dashboards
- Workspace templates
- Plugin marketplace
