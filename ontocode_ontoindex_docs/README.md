# OntoIndex + OntoCode Documentation Package

This package contains the planning and specification documents for two related products:

1. **OntoIndex** — a Rust-native embedded ontology index/query/analysis engine.
2. **OntoCode** — a VS Code ontology engineering workbench intended to become a Protégé replacement for ontology-as-code workflows.

## Documents

- `PLAN.md` — combined product plan
- `SPEC.md` — combined technical specification
- `ARCHITECTURE.md` — end-to-end architecture
- `ROADMAP.md` — milestone roadmap from v0.1 to v1.0
- `PLUGIN_SPEC.md` — plugin system design
- `LSP_SPEC.md` — language server protocol design
- `REASONER_SPEC.md` — reasoner adapter design
- `SEMANTIC_DIFF_SPEC.md` — semantic ontology diff design
- `UI_WIREFRAMES.md` — text-based VS Code UI wireframes
- `MVP_BACKLOG.md` — v0.1/v0.2 implementation backlog
- `v1.0_BACKLOG.md` — full v1.0 implementation backlog
- `adr/` — architecture decision records
